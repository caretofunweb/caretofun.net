📖 How to Use
English / Roman Urdu
اردو
  • 1
    Click 🎙️ Mic button to start — phir bolo, text khud likha jayega! Allow microphone if the browser asks. | Agar mic permission poochhe toh "Allow" karein.
  • 2
    Choose language: Urdu (ur-PK) or English (en-US) from the dropdown. | Zabaan chunein: Urdu ya Angrezi.
  • 3
    Press 🛑 Stop (same mic button) to stop recording. | Rokne kay liye dobara mic button dabaein.
  • 4
    Click 💾 Download button — a popup opens. Choose your format (TXT / DOCX / PDF) and then click "Download Now". | Download button dabaein, format chunein, phir Download Now.
  • 5
    Use 📋 Copy to copy text to clipboard. | Copy button se text clipboard pe copy ho ga.
  • 6
    Undo / Redo arrows let you go back or forward. Max 6 history states saved. | Undo/Redo se pechle version wapis laa sakte hain.
  • 7
    Select text in the box and press B / I for bold or italic markers. | Text select kar ke B ya I button se bold/italic lagaein.
  • 8
    Toggle 🌙 / ☀️ Theme for dark or light mode. | Theme button se dark/light mode tabdeel karein.
  • 9
    Note: This app uses Google's Web Speech API — internet connection zaroor chahiye. Works best in Google Chrome.
